Lab 2 - Pod 

1. Create basic Pod definition
2. Use kubectl to create the pod on cluster
3. Expose it
4. Access it (use the exposed port!)
5. Delete the service and pod to shut down

kubectl create -f pod-basicapp.yml
kubectl get pods
kubectl describe pod
curl localhost:3000
kubectl expose pod basicapp --type=NodePort --name basicapp
kubectl get services
kubectl delete service basicapp
kubectl get pods

kubectl delete pod basicapp
